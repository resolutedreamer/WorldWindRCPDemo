The textabbr.txt, Text Abbreviations Assignment File, defines a
set of abbreviations to replace values or value descriptions
which are actually stored in the specified label attribute. 
The first row contains the Abbreviation Index Identifier followed
by a colon(:).  Each subsequent row, until the next row with an ID
and colon is encountered, defines two pipe-delimited (|) entries
with the attribute value and the abbreviation to be used.
ID's are related to attributes as follows - 1:(ccc) 2:(psc) 3:(mcc) 4:(sst)
